# 🎮 Neon Boxes

**Neon Boxes** is a fast-paced first-person arcade shooter built using **Unity and C#**.  
Set in a glowing neon environment, the game challenges players to react quickly and make strategic decisions while shooting different types of boxes that affect score and survival time.

---

## 🎯 Game Concept

The game is based on a simple idea:

> **Every action has a consequence.**

Players must balance **scoring points** and **staying alive** while a countdown timer continuously runs.

---

## 🕹 Gameplay Mechanics

Three types of boxes appear during gameplay:

- **Pink Boxes** – Give **+5 score**
-  **White Boxes** – **Increase remaining time**
-  **Blue Boxes** – **Reduce remaining time**

The game ends when the **timer reaches zero**.
